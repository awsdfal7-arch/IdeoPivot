# 思政智题云枢

这是一个基于 `PyQt6` 的桌面应用，用来整理思政选择题题库。当前项目以“本地 SQLite 总库 + 开始页菜单导入/导出”为核心流程，支持文档导题、库内查重、AI 补解析，以及 Markdown / xlsx 导入导出。

## 当前功能

- 开始页题库浏览：
  - 左侧按 `必修三 -> 第x课 -> 第x框` 展示库内层级
  - 右侧表格展示题目、选项、答案、解析
  - 支持双击条目打开对话框并修改常用字段
  - 支持视图菜单控制显示哪些列，且列显示状态会持久化
- 导入弹窗中的确认导入资料页：
  - 左侧上方支持选择/拖拽 `docx`
  - 左侧下方显示导入提醒，并检查 `Word` 文档中是否包含图片/表格
  - 右侧提供文档预览：`docx` 转 PDF 后预览
- AI 导入 `docx`
- AI 导入详情页：
  - 表格列为 `Deepseek耗时 / Kimi耗时 / 千问耗时 / Deepseek返回 / Kimi返回 / 千问返回 / 对比结论`
  - 展示 `DeepSeek / Kimi / 千问` 三模型返回结果与对比结论
  - 支持红黄高亮区分少数派结果与空返回
- 确认导入资料页直接填写当前批次资料的 `版本 / 层级`
- 程序设定支持控制导入流程是否默认执行库内查重
- 程序设定支持控制导入文档时是否自动生成解析（`自动生成 / 不生成`）
- 导入流程不再保留单独的“查重选项页 / 解析选项页”，是否进入后续步骤由程序设定直接决定
- 库内查重：直接对当前总库 `.db` 做相似题比对，无需指定路径
- AI 生成解析：对缺失解析的题目批量生成解析并写回题库
- Markdown 导出：从开始页菜单栏导出当前选中层级的 Markdown
- xlsx 导入导出：
  - 支持从数据库字段表 `xlsx` 直接导入
  - 支持导出当前层级为 `xlsx`
  - 支持导出整体数据库表为 `xlsx`
- 导入预览使用项目内置 PDF 预览模块 `sj_generator/ui/pdf_preview`

## 并发设置

- 程序设定中提供统一并发数选择：`1 / 2 / 3 / 4 / 5`
- 程序设定中可设置默认题库保存位置，默认值为 `桌面/思政题库`
- 程序设定中使用一个可输入下拉框设置解析生成模型，格式为 `服务商 / 模型名`
- 解析生成模型默认值为 `DeepSeek / deepseek-reasoner`
- 解析生成模型每次修改后都需要先测试通过，才能保存程序设定
- 该设置同时作用于：
  - AI 导题时的题级并发
  - AI 生成解析时的批量并发
- 默认并发数为 `3`

## 模型配置

- 在“开始”页的“设置 -> 配置API”中维护模型配置
- AI 导题阶段需要：
  - `DeepSeek`
  - `Kimi`
  - `千问`
- AI 生成解析阶段可在程序设定中选择 `DeepSeek / Kimi / 千问`
- AI 生成解析阶段的模型名支持手动输入，默认使用 `deepseek-reasoner`
- 当选择 `DeepSeek` 时，解析生成使用其单独的解析模型配置
- 配置采用 OpenAI 兼容接口形式：`base_url / api_key / model`

详细说明见 [AI接口与提示词](docs/AI接口与提示词.md)。


```bash

```

## 使用流程

1. 在开始页确认 API 配置，并按需打开程序设定调整统一并发数
2. 浏览当前总库 `.db` 中的题目，必要时可直接双击编辑
3. 在开始页点击“导入 -> 从文档文件解析导入”，打开独立导入弹窗
4. 在弹窗中确认导入资料（支持拖拽补充文件），填写当前批次统一的 `版本 / 层级`，查看导入提醒与文档预览，然后执行 AI 解析
5. 在 AI 解析详情页查看三模型返回结果与对比结论
6. 按程序设定决定是否执行库内查重
7. 按程序设定决定是否自动执行 AI 补解析
8. 导入弹窗完成后自动写入当前题库 `.db`，回到开始页继续浏览，或在菜单栏导出 Markdown / xlsx

更完整的页面流程见 [向导流程](docs/向导流程.md)。

## 当前页面

- 主窗口当前保留 `登录` 与 `开始` 两页；资料导入使用独立导入向导弹窗。
- 主窗口不再内嵌导入页栈，导入相关页面仅在独立弹窗中注册与运行。
- `登录` 页目前主要是占位入口：
  - `本地登录` 直接进入开始页
  - `用户登录` 仅弹出失败提示，不是完整账号体系
- 页面名称如下：
  - `登录`
  - `开始`
  - `确认导入资料`
  - `AI 解析详情`
  - `库内查重（结果）`
  - `AI 生成解析`
  - `导入数据库成功`
- 其中部分页面按流程分支决定是否进入：
  - 从开始页菜单打开导入弹窗后，会依次进入 `确认导入资料`、`AI 解析详情`
  - 程序设定启用库内查重时会进入 `库内查重（结果）`
  - 程序设定启用自动解析生成时会进入 `AI 生成解析`

## 题库格式

- 当前题库主存储为本地 SQLite 数据库文件
- 数据库路径由“常规设定”中的默认题库保存位置决定
- 默认位置为 `桌面/思政题库/思政题库.db`
- 同时保留两类 `xlsx` 流程：
  - 标准题库表：用于传统 5 列题库读写
  - 数据库字段表：用于数据库记录导入导出
- 标准题库表头为：
  - `编号`
  - `题目`
  - `选项`
  - `答案`
  - `解析`

详细格式见 [数据结构与题库格式](docs/数据结构与题库格式.md)。
